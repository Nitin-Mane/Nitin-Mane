## Projects

### Smart Sterilizer Box for Mask and PPE Kits

The Smart Sterilizer Box for Mask and PPE Kits is a low-cost, portable solution designed to sterilize masks and personal protective equipment (PPE) kits. It utilizes UV-C light to effectively sterilize the items, making it particularly useful in low-resource settings such as hospitals. The project aims to enhance the health and safety of hospital staff and patients through safe and efficient sterilization processes.

[Project Repository](https://bit.ly/NitinSmartSterilizerBox)

### Peter Moss COVID-19 AI Research Project (Python)

The Peter Moss COVID-19 AI Research Project focuses on eXplainable Deep Neural Network (xDNN) classifiers developed by Professor Plamen Angelov and researcher Eduardo Soares PhD. These classifiers provide real-time predictions for medical support applications using intelligent networks. The project repository includes MATLAB and Python implementations of the xDNN architecture, enabling the utilization of xDNN on the edge for medical purposes.

[Project Repository](https://github.com/COVID-19-AI-Research-Project/xDNN/tree/master/Projects/Python/1)

### Automated Weed Detection and Removal in Agricultural Fields

The Automated Weed Detection and Removal project utilizes computer vision techniques to accurately detect and remove weeds in agricultural fields, leading to improved crop yields. The project leverages a Residual Network (ResNet) architecture and Auto Mixed Precision (AMP) for efficient and accurate classification of crop and weed images. By optimizing resources and minimizing environmental impact, the project contributes to sustainable agriculture practices.

[Project Repository](https://github.com/Nitin-Mane/Automated-Weed-Detection)

### Road Sign Detection and Alert

The Road Sign Detection and Alert project focuses on developing a real-time road sign detection system using a Raspberry Pi and a camera module. The system is designed to detect common road signs such as stop signs, speed limit signs, and yield signs, and triggers an LED alert upon detection. This project aims to improve road safety by providing timely notifications to drivers.

[Project Repository](https://github.com/Nitin-Mane/Road-Sign-Detection-and-Alert)
